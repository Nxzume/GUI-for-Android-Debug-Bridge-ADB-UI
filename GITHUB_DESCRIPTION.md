# GitHub Repository Description

## Short Description (for GitHub repo page)
A modern, user-friendly GUI application for Android Debug Bridge (ADB) operations on Windows. Built with PyQt6, featuring device management, app installation/uninstallation, DeGoogle functionality, screenshots, logcat viewing, and more. Available as a standalone executable or run from source.

## Longer Description (for README or About section)
A comprehensive ADB GUI tool for Windows that simplifies Android device management through an intuitive interface. Features include:

🔌 **Device Management** - Auto-detect and manage connected Android devices
📱 **App Management** - Install, uninstall, enable/disable apps with ease
🗑️ **DeGoogle** - Remove Google apps safely with categorized package lists (Simple/Custom modes)
📸 **Screenshots** - Capture device screenshots automatically
📋 **Logcat** - View real-time Android logs
💻 **Shell Commands** - Execute ADB shell commands directly
🌙 **Dark Mode** - Toggle between light and dark themes
📦 **Standalone Executable** - No Python installation required

Perfect for developers, power users, and anyone who wants to manage their Android device without using command-line tools.
